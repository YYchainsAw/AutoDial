# 自动宽带拨号程序

这个程序会自动监测网络连接状态,当检测到断网时自动尝试连接Netkeeper宽带。

## 使用前准备

### 1. 安装Python
- 下载并安装Python 3.x (https://www.python.org/downloads/)
- 安装时勾选"Add Python to PATH"

### 2. 创建宽带连接
在Windows中手动创建宽带连接:
1. 打开"控制面板" → "网络和共享中心"
2. 点击"设置新的连接或网络"
3. 选择"连接到Internet" → "宽带(PPPoE)"
4. 输入用户名:`202426704040@jxsd`
5. 输入密码:`704040`
6. 连接名称:**必须设置为"Netkeeper"**

## 使用方法

### 方式1: 手动运行
双击 `start_auto_dial.bat` 启动程序

### 方式2: 开机自启动(推荐)
1. 按 `Win + R` 打开运行窗口
2. 输入 `shell:startup` 并回车
3. 将 `auto_dial_silent.vbs` 的快捷方式复制到打开的文件夹中
4. 重启电脑测试

### 方式3: 直接运行Python脚本
```bash
python auto_dial.py
```

## 功能特点

✅ 自动检测网络连接状态(每30秒)
✅ 断网时自动拨号重连
✅ 连接失败自动重   
✅ 详细的日志记录(保存到auto_dial.log)
✅ 支持开机自启动
✅ 连续失败保护机制

## 日志文件

程序运行时会在同目录下生成 `auto_dial.log` 日志文件,记录所有操作。

## 停止程序

- 如果在命令行窗口运行,按 `Ctrl + C` 停止
- 如果是后台运行,在任务管理器中结束Python进程

## 注意事项

⚠️ 建议以管理员权限运行

## 故障排除

### 问题1: 提   "连接不存在"
- 确保已在Windows中创建名为"Netkeeper"的宽带连接

### 问题2: 拨号失败
- 检查用户名和密码是否正确
- 确保网卡驱动正常
- 尝试手动拨号测试

### 问题3: 检测不到断网
- 检查防火墙是否阻止了ping命令
- 确保网卡未被禁用
